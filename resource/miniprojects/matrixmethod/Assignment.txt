Write a class, Matrix, that has the following methods and properties:
Properties of its dimensions (rows, columns)
A two-dimensional array of integers
Constructor that sets the rows and columns of the Matrix
Relevant accessor and mutator methods
transpose method that returns the transpose of the Matrix
rotateArray method that returns the Matrix rotated by a multiple of 90
add method that returns the sum of the Matrix and its parameter. Assume that the dimensions for the Matrix and parameter are the same.
